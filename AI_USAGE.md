# AI Usage Declaration

This project, **AstroGem**, was developed with the assistance of advanced AI coding agents. 

## Summary of AI Contribution

- **Scaffolding and Setup:** AI was utilized to set up the React + Vite + TypeScript project structure.
- **UI/UX Design:** AI assisted in writing the custom Vanilla CSS framework, including the implementation of the glassmorphism layout, high-contrast dark/light mode toggle system, and the HTML5 canvas-based `CelestialBackground` starfield animation.
- **Astrology Engine Implementation:** The core logic mapping numerology numbers, zodiac approximations, and gemstone compatibility (friendly vs inimical planetary relationships) was synthesized and generated using AI's knowledge base of Vedic astrology.
- **Content Generation:** The descriptions, mantras, planetary attributes, and benefits for the 9 sacred gemstones (Navaratna) were collated and formatted by AI to ensure authentic and high-quality data.

All code and concepts were reviewed to align perfectly with the "Hamara Pandit" brand vision, ensuring a premium, professional, and mathematically coherent astrological web application.
